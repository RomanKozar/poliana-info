# Images structure

Place static photos in categorized folders:

- `public/images/branding/` - logo, favicon sources, brand assets
- `public/images/accommodation/` - hotels, apartments, rooms
- `public/images/spa/` - spa, wellness, pools
- `public/images/kids-camps/` - kids camps and activities
- `public/images/entertainment/` - events and attractions
- `public/images/blog/` - blog cover images
- `public/images/contacts/` - maps, office photos, contact visuals
- `public/images/gallery/` - general destination gallery
